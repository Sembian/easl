# EASL HCV Abstract Dashboard

Static dashboard for prioritising EASL HCV abstracts, served by a minimal Express app.

## Requirements

- Node.js 18+

## Install

```bash
npm install
```

## Run

```bash
npm start
```

Then open <http://localhost:3000>.

Override the port with `PORT=4000 npm start`.

## Files

- `2026_easl_abstracts_dashboard.html` — the full dashboard (data embedded)
- `server.js` — Express static server
- `package.json` — Node manifest
